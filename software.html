SOFTWARE

Que es el software?
El software es el conjunto de programas y rutinas que permiten al computador tener la habilidad de realizar funciones o tareas especificas,
el software es indispensable para la humanidad tanto como en el ambito administrativo como en el financiero,
facilitando sus operaciones.
